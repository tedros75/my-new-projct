
# Building AI course project

Final project for the Building AI course

## Summary
This is a Building AI course project. The idea is to create an AI tool that helps people plan their grocery shopping by suggesting what to buy based on their habits and reducing food waste.

## Background
Many people buy too much food and end up throwing it away. This project aims to help people make smarter shopping decisions and save money.

## How is it used?
The user inputs what they usually eat and what they already have at home. The AI suggests a shopping list and recipes based on this information.

## Data sources and AI methods
The project could use data about food consumption, recipes, and shopping habits.

## Challenges
The system needs accurate data and must adapt to different users’ preferences. It may also be difficult to predict what people actually want to eat.

## What next?
The project could be expanded into a mobile app with real-time suggestions and integration with grocery stores.

## Acknowledgments
This project was created as part of the Building AI course. Some ideas were inspired by online resources and examples.
